The 116 % MAWP default is based on ASME Section VIII allowances for vapor or gas relief when only one pressure-relief device (PRD) protects the system.
Here’s the logic:

Liquid service: Max accumulation = 10 % → 110 % MAWP.

Vapor/gas service (single PRD): Max accumulation = 16 % → 116 % MAWP.

Multiple PRDs: Allow 10 % each (so effectively still 110 % limit).
